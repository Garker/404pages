实现跳转到上一页的三类方法:

总述

主要是用于一些属性的设置和事件监听，如：

方法1：a标签的超链接里写js

注意：一定要写 javascript:，因为这是内嵌式js代码；可以不写window这个顶级对象。

返回上一页面 javascript:window.history.go(-1); 返回上一页面 javascript:history.go(-1); 返回上一页面 javascript:history.back(); 返回上一页面 javascript:history.back(-1); 返回上一页面 javascript:location.href = document.referrer;

方法2：a标签设置onclick()事件

return false禁止链接跳转到href的地址。建议用第一个，加上javascript:window.，go(-1)和back(-1)都可以，这里不一一列举。a标签设置onclick()事件,可以省略javascript:，因为这里面事件监听本来就是有js来完成的， return false禁止链接跳转到href的地址

javascript:window.history.go(-1);return false; 返回上一页面 javascript:history.go(-1);return false; history.go(-1);return false; history.go(-1);不加return false;无效 history.go(-1);return false; history.go(-1);不加return false;有效 history.back(-1);不加return false;有效 location.href = document.referrer; return false; history.go(-1);不设置href，会不显示a标签特征：变色，下划线；但是可以实现跳转。

方法3：input,button标签设置onclick()事件

可以省略window和 return false（因为这两个标签是没有超链接的，所以不存在禁用超链接）;
